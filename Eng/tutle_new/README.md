## Rectangle with a `turtle`  
```python
import turtle

t = turtle.Turtle()
t.shape("turtle")

t.forward(100) 
t.left(90) 

t.forward(50)
t.left(90)

t.forward(100)
t.left(90)

t.forward(50)
```  

### You can choose different colours with `t.color("color_name")`  
```python
import turtle

t = turtle.Turtle()
t.shape("turtle")

t.color("red") # Turtle color
t.forward(100)

t.color("blue")
t.forward(100)

t.color("tomato")
t.forward(100)

```  

### Let's create multicolour lines with gaps, using `t.up()` and `t.down()`  
```python
import turtle

t = turtle.Turtle()
t.shape("turtle")

t.color("green")
t.forward(20)
t.up()
t.forward(20)
t.down()

t.color("yellow")
t.forward(50)

t.up()
t.forward(20)

t.down()
t.color("blue")
t.forward(20)
```
